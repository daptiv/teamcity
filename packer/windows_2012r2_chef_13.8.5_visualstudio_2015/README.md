# VisualStudio AMI

Creates an EC2 AMI with Visual Studio 2015 pre-installed.

## Prereq's

* [Packer](https://www.packer.io/)

## Supported Platforms

* Windows 2012 R2

Usage
=====

```
$> docker-compose run --rm packer rake build
```
