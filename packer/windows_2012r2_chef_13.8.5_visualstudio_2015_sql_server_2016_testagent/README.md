# Test Agent AMI

Creates a Test Agent EC2 AMI using the
windows_2012r2_chef_13_8_5_visualstudio_2015_sql_server_2016 AMI as its base.

## Prereq's

* [Packer](https://www.packer.io/intro/)

## Supported Platforms

* Windows 2012 R2

Usage
=====

```
$> rake build
```
